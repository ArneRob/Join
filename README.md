Join — Task Management App
Join is a Kanban-style project management web app built with vanilla JavaScript and Firebase.

Features
Board view — drag-and-drop tasks across To Do, In Progress, and Done columns
Task management — create tasks with subtasks, priorities, deadlines, and assigned contacts
Contact management — manage team members and assign them to tasks
Summary dashboard — overview of upcoming deadlines and task counts
Authentication — sign up, log in, or continue as a guest via Firebase Auth
Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+)
Backend: Firebase Realtime Database & Authentication
Architecture: Client-side SPA, modular JS, no build step required
